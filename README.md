# 🍰 Пряничный Дом — Красноярск

Сайт премиальной кондитерской в Красноярске. Пряничные достопримечательности города, макарон, торты и авторский кофе на вынос.

## Что внутри

- 🏛 **Пряничные достопримечательности** — Часовня Параскевы Пятницы, Коммунальный мост, Театр оперы и балета, расписанные вручную королевской глазурью
- 🍰 **Макарон** — ассорти из 12 вкусов
- 🎂 **Торты** — «Розовый Исполин», «Фисташковый салон» и другие
- ☕ **Кофе на вынос** — флэт уайт с фисташкой, капучино с розой, латте с солёной карамелью
- 🛒 **Корзина** — добавление товаров, количество, итого
- 🚚 **Доставка** — по Красноярску от 3 000 ₽ бесплатно, за 90 минут

## Стек

- **React 19** + **TanStack Router** (file-based routing)
- **Vite 6** — сборка и dev-сервер
- **Tailwind CSS 4** — стили
- **TypeScript**

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Структура

```
src/
├── routes/          # Страницы (file-based routing)
│   ├── index.tsx    # Главная
│   └── collection.tsx # Каталог
├── components/      # UI-компоненты
├── lib/             # Утилиты, данные, корзина
├── assets/          # Изображения
└── styles.css       # Глобальные стили и дизайн-система
```

## Дизайн

Вдохновлён Ladurée и парижскими кондитерскими:
- Шрифты: Cormorant Garamond (заголовки), Jost (основной текст), Parisienne (акценты)
- Палитра: кремовый, пистачио, роза, шампанское золото
- Анимации: снежинки, пар от кофе, Ken Burns на герое
